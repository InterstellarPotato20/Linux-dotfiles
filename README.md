# Linux-dotfiles
Dotfiles modified for personal use. To be saved in ~/.config. Currently includes files for Openbox WM and tint2 panel.


## Edits in /openbox/rc.xml for keybindings
*Note: (W = Windows/Super key)*

W-q = firefox

W-e = nemo

W-t = xfce4-terminal

W-s = fsearch

W-f = ToggleMaximize

W-l = openbox --exit

W-x = xfce4-appfinder

W-c = mousepad
